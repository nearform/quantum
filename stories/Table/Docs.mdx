# Table

A table column is a vertical division within a table that organises and displays a specific type of data. Each column typically has a header at the top, describing the content it contains.

The Table component is built up of several components, the general structure of which is shown below.

```bash
└── Table
    ├── 1.TableHeader
    │   └── TableRow
    │       └── TableHead
    ├── 2.TableBody
    │   └── TableRow
    │       └── TableCell
    ├── 3.TableFooter
    │   └── TableRow
    │       └── TableCell
    └── 4.TableCaption
```
