# ControlLabel

A control label is a label for a form control component (e.g. switch, checkbox, radio button, etc).

## Example

```tsx
import { Switch, ControlLabel } from '@/index'

<ControlLabel htmlFor="fieldId" label="Label"  hintText="This is a hint">
  <Switch id="fieldId" value="value" />
</ControlLabel>
```
